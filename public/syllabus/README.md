# JEE Syllabus PDFs Directory

Place your official JEE syllabus PDF files in this directory:
- `JEE_Main_Syllabus_2026.pdf` (JEE Main Syllabus)
- `JEE_Advanced_Syllabus_2026.pdf` (JEE Advanced Syllabus)

When you push this repository to GitHub and deploy to Vercel, these files will be compiled into the static assets and will download instantly when users click the download buttons in your PrepTrack application!

### Dynamic Fallback Enabled:
PrepTrack has a built-in high-fidelity print-ready viewer. If these files are not present in the deployed environment, the application will automatically render a gorgeous, styled syllabus document with checkboxes and study guidelines that users can print or save directly as a clean PDF!
